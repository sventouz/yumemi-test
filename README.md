## local development

Before you start, you need to add `VITE_API_KEY` to `.env` file.  
You can get API key from here.  
https://opendata.resas-portal.go.jp/

commands to start.

```
npm install
npm run dev
```

## lint and format

```
npm run lint
npm run format
```
